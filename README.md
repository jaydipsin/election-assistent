# 🗳️ Matdan – Smart Voting Awareness Platform

Matdan is an educational web platform designed to guide citizens—especially first-time voters—through the complete voting process in India. It simplifies complex procedures and provides step-by-step assistance, ensuring users understand how to register, prepare, and vote confidently.

> ⚠️ This project is strictly for educational purposes and does not support or promote any political party or agenda.

---

## 🚀 Problem Statement

Many citizens, especially first-time voters, face confusion around:

* Voter ID registration
* Voting procedures
* Rules and eligibility criteria
* What to expect at a voting booth

This lack of clarity leads to hesitation and low participation.

---

## 💡 Solution

Matdan solves this by providing:

* 📘 Step-by-step voting guidance
* 🪪 Voter ID registration assistance
* 📖 Verified references from official sources
* 🤖 AI-powered chatbot for instant query resolution

---

## 🧠 Key Features

* **AI Chatbot (Gemini 3 Flash Preview)**
  Helps users ask questions about voting, rules, and procedures in real-time.

* **Step-by-Step Voting Guide**
  Clear instructions on what to do before, during, and after voting.

* **Voter ID Registration Help**
  Guidance with official references for applying for a voter ID.

* **Trusted Information Sources**
  Links and references from official platforms to ensure accuracy.

* **Beginner-Friendly UI**
  Designed for simplicity so anyone can understand the process easily.

## 🛠️ Installation & Setup

```bash

git clone https://github.com/your-username/matdan.git

# Navigate to project
cd matdan

# Install dependencies
npm install

# Run frontend (Angular)
ng serve

# Run backend (Node)
npm start
```
